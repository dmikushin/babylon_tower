Вавилонская Башня

Правила игры:

  1. Цель игры - переместить все диски с первого стержня на последний.
  2. За один ход можно переместить только один диск.
  3. Диск можно положить только на больший диск или на пустой стержень.

Инструкция: Кликните сначала на стержень, с которого хотите взять диск, а затем на стержень, куда хотите его переместить.
