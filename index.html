Вавилонская Башня

Ходов: 0
Используйте клавиши 1-3 для выбора стержней (сначала источник, затем назначение)

Правила игры:

  1. Цель игры - переместить все диски с первого стержня на последний.
  2. За один ход можно переместить только один диск.
  3. Диск можно положить только на больший диск или на пустой стержень.

Инструкция: Коснитесь сначала стержня, с которого хотите взять диск, а затем стержня, куда хотите его переместить.
